Programação na Web

Na Escola do Comércio de Lisboa, na disciplina de Tecnologia 1 no 1ºATIS, iremos aprender Programação Web 😅 e abordaremos os princípios básicos associados ao desenvolvimento para a Web, apenas do lado <1>front-end, lado visual. Também sabemos que existe um lado back-end, que é o lado do servidor, mas não iremos abordar esse lado.

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar todos os conhecimentos, serão feitos alguns laboratórios.
